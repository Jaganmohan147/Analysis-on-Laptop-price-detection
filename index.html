   Analysis on
Laptop Price Detection

Abstract

Laptop prices depend on various factors like brand, processor, RAM, storage, and display size. Understanding these factors helps in predicting laptop prices accurately. This project analyzes a laptop dataset using Pandas Profiling. The goal is to find patterns in specifications and pricing. This analysis helps buyers choose the right laptop and sellers set better prices.

Introduction

Laptop prices vary based on performance, features, and brand reputation. Predicting laptop prices helps both customers and manufacturers. In this project, we analyze a laptop dataset to find important factors affecting price. We use Pandas Profiling to explore trends and patterns in laptop pricing. This helps in
understanding which specifications influence price the most

Literature Survey

Many researchers have studied laptop price prediction using data science. Some use machine learning models to predict prices based on features. Others analyze market trends to find price variations. Traditional price estimation relies on manual comparison, but automated analysis makes the process faster and more accurate. Studies show that Exploratory Data Analysis (EDA) helps in understanding how laptop features impact price

EDA

EDA helps in analyzing laptop specifications and pricing. Using Pandas Profiling, we can find patterns in processor type, RAM, storage capacity, screen size, battery life, and price. High-end laptops usually have powerful processors, more RAM, and better displays. EDA helps to compare different laptop brands
and models.

Technology Used
